## Objetivo

Explorar os diferentes papéis profissionais na área de cibersegurança
e perceber qual o caminho mais adequado para os meus objetivos.

## Papéis Explorados

### Penetration Tester (Red Team)
Simula ataques a sistemas para encontrar vulnerabilidades.
Trabalha com autorização do cliente para testar a segurança antes
que um atacante real o faça.

Ferramentas típicas: Kali Linux, Metasploit, Burp Suite, Nmap

### Security Analyst (Blue Team)
Monitoriza sistemas, analisa alertas e responde a incidentes.
Trabalha em SOCs e usa SIEMs para ter visibilidade sobre a infraestrutura.

Ferramentas típicas: Splunk, Wazuh, Wireshark, ELK Stack

### Security Engineer
Constrói e mantém a infraestrutura de segurança da organização.
Implementa firewalls, sistemas de deteção de intrusão e políticas de segurança.

### Malware Analyst
Analisa software malicioso para perceber como funciona e como defendê-lo.
Trabalha em ambientes isolados (sandboxes) para dissecar malware com segurança.

### Digital Forensics Investigator
Investiga incidentes após ocorrerem, recupera evidências digitais
e reconstrói o que aconteceu durante um ataque.

## Reflexão Pessoal

alinhado com o meu objetivo, pois permite usar o conhecimento ofensivo
para melhorar a defesa.
